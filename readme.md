# Mein Moers Backend

[![Events Package Tests](https://github.com/LambdaDigamma/mm-events/actions/workflows/run-tests.yml/badge.svg)](https://github.com/LambdaDigamma/mm-events/actions/workflows/run-tests.yml)

Backend API service for the Mein Moers mobile applications, providing information about events, parking, rubbish collection, organizations, and other services in the city of Moers, Germany.

### API Versions

The API provides two major versions:
- **v1**: Original API endpoints for core features (events, parking, rubbish)
- **v2**: Enhanced API with additional features (organizations, polls, admin)

Both versions are currently supported. See the [API Versioning and Deprecation Policy](docs/api-documentation.md#api-versioning-and-deprecation-policy) for details.
